# v0.2.3 发布说明

## 代码审查修复

- AAAA + IPv6 节点校验闭环（按分组 DNS 记录类型校验公网 IP）
- CLI `status` / `config-check` 使用 `NewDNSProvider`，Route53 部署诊断准确
- Agent `status` 显示真实 `traffic_mode`
- 安装脚本 URL 随 `version.Version` 动态生成（不再硬编码 v0.1.0）
- Web 状态面板：未配置 `QDR_STATUS_READONLY_TOKEN` 时仅 localhost 可访问

## 运维增强

- Route53：`dns_provider` 写入 settings，诊断区分 Cloudflare Token / Route53 AWS 凭证
- Agent 上报 `traffic_mode` 与节点配置不一致时记录 `agent_traffic_mode_mismatch` 告警
- Master `run` 优雅关闭（首个 worker 退出后 cancel + 10s 等待其余 worker）
- `telegram-run` 启动时输出废弃警告
- 删除孤立 `internal/system/systemd.go`

## 安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ike-sh/quota-dns-router/v0.2.3/scripts/install-master.sh)
```
