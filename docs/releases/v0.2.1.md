# v0.2.1 发布说明

**发布日期**：2026-06-08

## 新功能

- **只读 Web 状态面板**：`GET /`、`/status` 与 `GET /api/status`
- **Telegram 只读观察者**：`QDR_TELEGRAM_OBSERVER_IDS` 可查看状态，不可修改
- **DNS 服务商工厂**：`QDR_DNS_PROVIDER=cloudflare`（为第二服务商预留）

## 修复与加固

- join 限流仅在可信反代后信任 `X-Forwarded-For`
- join 限流 map 自动清理过期条目
- Release CI：arm64 交叉编译不再在 amd64 runner 上执行二进制

## 配置示例

```env
QDR_STATUS_READONLY_TOKEN=replace-me
QDR_TELEGRAM_OBSERVER_IDS=111222333
QDR_DNS_PROVIDER=cloudflare
```

bash <(curl -fsSL https://raw.githubusercontent.com/ike-sh/quota-dns-router/v0.2.1/scripts/install-master.sh)
