# v0.2.3.1

文档与 smoke 补丁版本（功能与 v0.2.3 相同）。

## 变更

- **README**：同步 Route53、只读 Web 面板、AAAA 记录；修正「仅 Cloudflare」等过时描述
- **smoke.sh**：版本号、`/healthz`、`/readyz`、本机 `/api/status` 验收
- **文档**：生产部署清单、gh CLI Release 指南、Smoke 验收清单

## 继承自 v0.2.3 的核心修复

- AAAA + IPv6 节点校验闭环
- CLI 使用 `NewDNSProvider`（Route53 诊断准确）
- Agent `status` 显示真实 `traffic_mode`
- 安装 URL 动态版本号
- Web 状态面板：无 Token 时仅 localhost
- `traffic_mode` 不一致 Telegram 告警
- Master 优雅关闭

## 安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ike-sh/quota-dns-router/v0.2.3.1/scripts/install-master.sh)
```

## Smoke

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/ike-sh/quota-dns-router/v0.2.3.1/scripts/smoke.sh) master
```
