# v0.2.3.5 发布说明

DNS 漂移修复、离线切换补全、Join 安全与 UI 文案。

## 修复

- **ResolveCurrentNode**：有 DNS 配置时优先查 DNS，不再盲目信任过期 `current_node_id`；漂移时自动同步。
- **从未上报节点**：`CreatedAt` 超过 `agent_offline_seconds` 视为离线，触发 failover。
- **noTargetMessage**：使用 `selectTargetCandidates` 真实候选数。
- **Join 码**：生成新码前删除同节点未使用旧码。
- **UI 文案**：`dnsRecordLabel` 按 A/AAAA 动态显示；setup 概览用「DNS 记录」。
