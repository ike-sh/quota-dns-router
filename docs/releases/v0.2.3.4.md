# v0.2.3.4 发布说明

流量采样、Route53 与 Join 安全加固。

## 修复

- **流量计数器回绕**：网卡计数器归零（重启）时 `computeDelta` 返回 0，避免虚假流量突增触发切换。
- **Route53 SetIdentifier**：从合成 `recordID`（`name/TYPE#setId`）解析加权路由标识，避免 upsert 错记录集。
- **Join 原子核销**：`RedeemJoinCode` 使用条件 `UPDATE` 原子标记已使用，防止并发双花。
- **Agent Token 轮换**：重新 join 时吊销旧 Token，仅保留最新凭证有效。
